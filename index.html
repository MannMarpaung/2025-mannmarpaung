My name is Zaid Hilman Al-Farisi Marpaung. I’m a Software Engineer in training and currently studying at IDN Boarding School Sentul. My coding journey started in high school when I took an online Python course offered by my dad. Since then, I’ve been passionate about programming—especially web development—using technologies like HTML, CSS, JavaScript, PHP, and frameworks like Laravel, Bootstrap, and Tailwind.

For me, programming is a way to express ideas, and I want to keep growing in it because it’s something that truly makes me happy.
